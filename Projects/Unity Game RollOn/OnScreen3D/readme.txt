RollOn Game:

To run this program you just need a keyboard and mouse. 
Un-zip the file.
Then find the .exe file and double click it. 
It is recomended that you use the windowed option so that you can close the program when finished playing.
Also, when the configuration screen opens, please use 1152x964 and quality either good or beautiful.

Thank you!
